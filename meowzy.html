Meowzy

Meowzy – A dedicated cat-focused website that provides detailed information on cat breeds, traits, and behaviors. It features engaging and informative articles such as How Cats Calm Us, Decoding Your Cat’s Secret Signals, The Ultimate Starter Guide: So You Just Got a Cat?, and Why Cats Can’t Resist Boxes, aimed at helping cat owners better understand and bond with their pets.

Goals / Purpose

The goal of Meowzy is to create an engaging, resource-rich platform for cat enthusiasts. It aims to educate users about cat behavior, care tips, and breed-specific characteristics, helping pet owners make informed decisions and build stronger connections with their feline companions.

Tools & Technologies Used

1. Frontend: HTML5, CSS3, JavaScript, Bootstrap
2. Backend: Node.js, Express.js (if applicable)
3. Database: MongoDB / MySQL (if applicable)
4. Design: Figma for UI/UX mockups
5. APIs: Google Maps API, Traffic Data API (for real-time updates)

Process / Development Steps

1. Planning – Researched cat-related topics and defined the target audience.
2. Design – Created wireframes and UI mockups in Figma to establish a clean and visually appealing layout.
3. Content Creation: Wrote informative and SEO-optimized articles on cat breeds, behaviors, and care tips.
4. Development: Built responsive web pages using HTML, CSS, and WordPress with custom styling.
5. Testing: Checked for responsiveness, accessibility, and cross-browser compatibility.
6. Deployment: Hosted the site on a live server and connected a custom domain.

Challenges & Solutions

1. Challenge: Organizing large amounts of informational content without overwhelming the user.
Solution: Implemented categorized navigation and a search bar for easy content discovery. 2. Challenge: Ensuring a visually engaging design while keeping it lightweight.
Solution: Used optimized images and minimalistic layouts to maintain fast load times.
