Cosmany

Cosmany – An online cosmetic store offering a diverse selection of beauty products, including makeup, nail care, and hair products. Designed with a clean, user-friendly interface, it allows customers to easily browse, compare, and purchase items while enjoying a seamless online shopping experience.

Goals / Purpose

The goal of Cosmany is to create an accessible and visually appealing e-commerce platform for beauty enthusiasts. It solves the problem of scattered cosmetic product options by offering a centralized store with curated, high-quality selections and a smooth purchasing process.

Tools & Technologies Used

1. HTML5 – For structure and semantic content
2. CSS3 – For styling, responsive design, and animations
3. JavaScript – For interactivity and dynamic content updates
4. Bootstrap – For layout and responsiveness
5. Figma – For wireframing and UI design
6. WordPress + WooCommerce (if implemented for e-commerce functionality)

Process / Development Steps

1. Planning – Defined target audience, product categories, and key features.
2. Design – Created wireframes and UI mockups in Figma to visualize layout and style.
3. Development – Built front-end with HTML, CSS, Bootstrap; implemented product listing and shopping cart features.
4. Integration – Connected WooCommerce (if WordPress) or a custom cart system for purchases.
5. Testing – Checked for cross-browser compatibility, responsive design, and checkout process flow.
6. Deployment – Uploaded files to hosting, configured domain, and optimized for speed and SEO.

Challenges & Solutions

1. Challenge: Ensuring the website looked appealing across different screen sizes.
Solution: Implemented a mobile-first responsive design using Bootstrap’s grid system.
2. Challenge: Managing product images and loading speed.
Solution: Used optimized image formats (WebP) and lazy loading to improve performance.
3. Challenge: Creating a smooth and secure checkout process.
Solution: Integrated a secure payment gateway and SSL encryption.
